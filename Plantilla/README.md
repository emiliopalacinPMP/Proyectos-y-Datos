# Edición #1: Automatización de métricas de Valor Ganado (EVM)
**Newsletter: Proyectos & Datos | Por: Emilio Palacín, PMP®**

## Descripción
Este repositorio contiene los activos técnicos de la simulación presentada en la primera edición del newsletter. El objetivo es demostrar cómo automatizar el análisis de desempeño de un proyecto integrando cronogramas y costos mediante Power BI.

## Contenido
- `/data`: Archivos CSV que simulan la estructura de un proyecto real.
- `/scripts`: Fórmulas DAX listas para ser utilizadas en modelos de datos.
- `/dashboard`: Archivo .pbix de referencia con la visualización terminada.

## Cómo usar este repositorio
1. Descargue el archivo `.pbix` de la carpeta `/dashboard`.
2. Si desea replicar el modelo, utilice los archivos en `/data` y conéctelos a Power BI.
3. Copie las medidas del archivo `/scripts/medidas_evm.dax` para crear sus propios indicadores.
