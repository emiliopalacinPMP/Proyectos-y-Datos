# ⚙️ Motor EVM para Control de Proyectos - Edición 1

Este repositorio es la fase cimiento de un ecosistema diseñado para cerrar la brecha entre la ingeniería de campo y la analítica directiva. Aquí, los estándares del **PMI (PMP)** se encuentran con la potencia de **Python**.

## 📊 Propósito de la Simulación
Se analiza una **Subestación Eléctrica de 115kV** al Mes 5 de ejecución. El motor inyecta hechos reales de obra (niveles freáticos, retrasos en procura) para validar la capacidad predictiva de los algoritmos de **EVM**.

## 🚀 El Motor a su Disposición
Se ha construido un script modular (`PD_ed1.ipynb`) que permite al usuario:
1.  **Auditar la Lógica:** Comprender cómo se calculan el EAC y el VAC bajo condiciones de bajo rendimiento.
2.  **Alimentación con Datos Propios:** El motor está diseñado para recibir archivos de avance externos y generar reportabilidad ejecutiva al instante.
